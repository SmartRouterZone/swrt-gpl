# Mi AC2100 Port Notes

## Summary

Mi AC2100 is adapted from the existing `SWRT-RM-AC2100` support. The port also used `lukasz1992/asuswrt-xiaomi` as a reference for Xiaomi/RM-AC2100 board naming and layout decisions.

## Build

```sh
cd release/src-mtk-mips
make swrt-mi-ac2100
```

Enable WireGuard for a temporary build with:

```sh
make swrt-mi-ac2100 WIREGUARD=y
```

Optional tools can be enabled through the existing make variables or config fragments used by this tree. Keep `.config` consistent with the selected model; if the build reports a model mismatch, remove the stale `.config` and build the target again.

## Build Fixes Applied

- Added the missing `flash_mtd.o` rule for the MI AC2100 utility build.
- Added the include path needed by `flash_mtd.c` to find `mtd-abi.h`.
- Fixed the `nvram` link path/order so `-lshared` resolves in the container toolchain.
- Avoided placeholder prebuild files; model prebuild entries must be real files or valid symlinks to compatible prebuilt directories.

## Notes

The build environment requires the MIPS MTK toolchain from the project README. The current notes cover build validation only; flashing and runtime behavior still need hardware testing.
